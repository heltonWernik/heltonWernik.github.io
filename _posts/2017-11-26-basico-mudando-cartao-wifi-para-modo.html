Para alguns ataques é necessário estar com o cartão wifi no modo Monitor.



Não são todos os cartões wifi que funciona, veja neste site quais funciona:
https://null-byte.wonderhowto.com/how-to/buy-best-wireless-network-adapter-for-wi-fi-hacking-2017-0178550/.

Por padrão os cartões sempre estão em modo Managed  que só recebem os pacotes que forem enviados ao nosso MAC address, porém com o modo monitor poderemos receber todos os pacotes no alcance de nosso wifi

Vamos aprender três métodos para mudar, pois algumas vezes ele não funciona, se fizer um e não funcionar, desligue o cartão , conecta novamente e muda de outra forma.

1. airmon-ng

Usando esse método comecaremos com o comando básico para conhecer qual é a nossa placa:

> airmon-ng
> airmon-ng start wlan0


Pronto, para usar a interface devemos usar o nome que ficou com a mudança , nesse caso mon0

Vamos verificar:

> iwconfig mon0


Quando mudamos para o modo monitor ele perde a conexão pois estará recebendo todos os dados.

Para voltar ao modo normal

> airmon-ng stop mon0


2. Mudando Manualmente

Alguns cartões não aceitam a mudança com o airmon-ng por isso vamos aprender outro método.

> iwconfig wlan0
> ifconfig wlan0 down
> iwconfig wlan0 mode monitor
> ifconfig wlan0 up
> iwconfig wlan0


Nesse método a interface wlan0 será o modo monitor.

3. Outro método com o airmon-ng

> airmon-ng
> ifconfig wlan0 down
> airmon-ng check kill
> airmon-ng start wlan0
