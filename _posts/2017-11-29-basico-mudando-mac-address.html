Mudando o MAC Address
MAC é a sigla de Media Access Control, ou seja, o Endereço MAC nada mais é que o endereço de controle de acesso da sua placa de rede. É um endereço único, com 12 dígitos hexadecimais, que identifica sua placa de rede em uma rede. De posse dos números e letras que formam os endereços MAC dos computadores da sua rede, você não precisa sequer definir uma senha para o Wi-Fi. Apenas computadores com o endereço autorizado poderão acessar a rede. O roteador não distribuirá endereços de IP para dispositivos sem o MAC cadastrado no sistema.



Podemos mudar nosso MAC para ser igual a um dos cadastrados no roteador e termos acesso.

Comandos para mudar:

Usando uma ferramenta chamada machanger
Para ver qual a rede que quer mudar
> iwconfig
Desligar ela
> ifconfig wlan0 down
Vamos usar uma ferramenta chamada macchanger
> macchanger --help
> macchanger --random wlan0
Depois devemos ligar novamente o cartão com o comando
> ifconfig wlan0 up


Mudando Manualmente
Caso esteja usando um sistema que não tem a ferramenta macchanger ou que ele não está funcionando podemos mudar ele manualmente com o ifconfig (que existe em todos os sistemas)

ifconfig para ver as interfaces
> ifconfig
> ifconfig wlan0 down
> ifconfig wlan0 hw ether 00:11:22:33:44:55
hw = hardware
ether = MAC address
00:11:22:33:44:55 = o MAC que quer mudar
> ifconfig wlan0 up
para verificar
> ifconfig wlan0
